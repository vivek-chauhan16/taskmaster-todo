TaskMaster - keep it done

TaskMaster is a task management web application built with Django to manage daily tasks efficiently. Users can add new tasks, edit existing ones, mark tasks as completed, and delete tasks when no longer needed.

The dashboard displays task statistics such as total tasks, pending tasks, and completed tasks, providing a clear overview of work progress.

This project is created with a focus on clean design, practical functionality, and real-world usage.
